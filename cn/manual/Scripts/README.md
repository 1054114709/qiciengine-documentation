# 脚本依赖管理
项目工程中常有脚本依赖的关系，A脚本依赖于B脚本，此时需要确保B脚本优先载入。  
__注意：这里所说的依赖，指的是脚本加载的依赖（可能并非逻辑依赖）__  

设置方法是在Project面板中随意选择一个脚本，在Inspector面板点击“Edit”按钮，通过拖动脚本位置建立加载的依赖关系：  
![](images/drag.png)  

* 脚本依赖列表中，靠前的优先加载，类似于grunt规则
* 有设置依赖的脚本优先加载，未设置依赖的脚本最后面加载
* 支持设置文件夹脚本依赖
