# qc.SoundManager

## 描述
游戏声音管理器，可以使用game.sound进行访问。
可通过 [game.add.sound](../gameobject/sound) 方法创建一个 [qc.Sound](../gameobject/CSound.md) 对象。

## 变量
| 变量名 | 类型 |作用 |
| ------------- |-------------|-------------|
| mute | boolean | 是否静音 |
| volume | number | 音量设置 |
