# ObjectLayer
inherit [qc.Node](CNode.md)

## 描述
瓦片地图的对象层

## 属性
| 变量名         |   类型      |  作用        |
| ------------- |-------------|-------------|
| tilemap | [qc.Tilemap](CTilemap.md) | 地图层所属的地图 |
| layerIndex | int | 地图层在所属地图中索引 |
| scrollXRatio | number | 地图层水平滚动的速率 |
| scrollYRatio | number | 地图层竖直滚动的速率 |
