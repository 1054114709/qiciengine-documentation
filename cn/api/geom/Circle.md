# Circle

## 描述
描述一个圆

## 构造函数
````
	var circle = new qc.Circle(0, 0, 1);
````

## 变量
| 变量名        |   类型       |  作用           |
| ------------- |-------------|-------------|
| x | number | 圆心X坐标 |
| y | number | 圆心Y坐标 |
| diameter | number | 直径 |
| radius | number | 半径 |

## 方法调用范例
````
	// 计算距离
	circle.distance({ x: 5, y: 6});
````
