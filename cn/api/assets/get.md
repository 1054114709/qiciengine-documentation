# [qc.AssetUtil](AssetUtil.md).get

## 原型
* get(string url, function onload, function onerror)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| url | string | 请求地址 |
| onload | function | 成功请求的回调，原型为：onload(data) |
| onerror | function | 请求失败的回调，原型为：onerror(XMLHTTPREQUEST xhr) |

## 说明
GET 请求数据
