# [qc.Math](README.md).random

## 原型
* number random(min, max)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| min | number | 最小值 |
| max | number | 最大值 |
| 返回值 | number | [min,max] 范围的随机值 |

## 说明
在[min,max]中取得一个随机值

## 范例
````
    // 返回 1
    var ret = game.math.random(1,4);

    // 返回 3
    var ret = game.math.random(1,4);
````
