# [qc.Math](README.md).linear

## 原型
* number linear(p0, p1, t)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| p0 | number | 初值 |
| p1 | number | 终值 |
| t | number | 比例 |
| 返回值 | number | 返回线性插值 |

## 说明
计算线性插值

## 范例
````
    // 返回 2.8
    var ret = game.math.linear(1,10,0.2);
````
