# [qc.arcade.RigidBody](../RigidBody.md).velocityFromRotation

## 原型
* booean velocityFromRotation(Rotation, speed, point)

## 参数
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| Rotation | number | 旋转弧度 |
| speed | number | 移动速度 |
| point | qc.Point | 水平和垂直方向的速度 |

## 返回值
水平和垂直方向的速度

## 说明
根据弧度和速度，得到水平和垂直方向的速度
