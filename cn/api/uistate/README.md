# qc.UIState
组件的交互状态：
````
	qc.UIState = {
	    /**
	     * @property {number} NORMAL - 正常状态
	     * @static
	     */
	    NORMAL : 0,

	    /**
	     * @property {number} PRESSED - 按下状态
	     * @static
	     */
	    PRESSED : 1,

	    /**
	     * @property {number} DISABLED - 不可用
	     * @static
	     */
	    DISABLED : 2
	};
````
