# [qc.Node](CNode.md).getScripts

## Prototype
* Array getScripts(string script)

## Parameters
| Parameter | Type | Description |
| --------- | --------- | --------- |
| script | string | The className of the script. |

## Description
Returns all components in the node.

## Sample
````
    node.getScripts('qc.TweenPosition');
````
