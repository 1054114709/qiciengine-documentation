# [qc.Node](CNode.md).setStretch

## Prototype

## Parameters
| Parameter | Type | Description |
| --------- | --------- | --------- |
| left | number | Distance left. |
| right | number | Distance right. |
| top | number | Distance top. |
| bottom | number | Distance bottom. |
