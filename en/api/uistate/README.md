# qc.UIState
The states of the object.
````
	qc.UIState = {
	    /**
	     * @property {number} NORMAL - the normal state
	     * @static
	     */
	    NORMAL : 0,

	    /**
	     * @property {number} PRESSED - the object is pressed.
	     * @static
	     */
	    PRESSED : 1,

	    /**
	     * @property {number} DISABLED - the object is disable.
	     * @static
	     */
	    DISABLED : 2
	};
````
