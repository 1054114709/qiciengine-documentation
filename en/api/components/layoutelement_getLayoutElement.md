# [qc.LayoutElement](LayoutElement.md).getLayoutElement

## Prototype
* object getLayoutElement(node)

## Parameter
| Parameter | type | Description |
| ------------- | ------------- | -------------|
| node | [qc.Node](../gameobject/CNode.md) | node |

## Description
Get the layout of a cell

## Example
return the following format:
````
    {
        ignoreLayout : NaN,
        minWidth : NaN,
        minHeight : NaN,
        preferredWidth : NaN,
        preferredHeight : NaN,
        flexibleWidth : NaN,
        flexibleHeight : NaN,
    };
````
