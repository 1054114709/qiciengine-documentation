# [qc.Math](README.md).degToRad

## Prototype
* number degToRad(degrees)

## Paramters
| Paramter | Type | Description |
| ------------- | ------------- | -------------|
| degrees | number | Angle in degrees.     |
| return | number | Angle in radians.    |

## Description
Convert degrees to radians.

## Example
````
    // return Math.PI
    var ret = game.math.degToRad(180);
````
