# [qc.Math](README.md).radToDeg

## Prototype
* number radToDeg(degrees)

## Paramters
| Paramter | Type | Description |
| ------------- | ------------- | -------------|
| degrees | number | Angle in radians.      |
| return | number |  Angle in degrees  |

## Description
Convert radians to degrees .

## Example
````
    // return 180
    var ret = game.math.radToDeg(Math.PI);
````
