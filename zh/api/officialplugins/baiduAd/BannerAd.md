# qc.Plugins.BannerAd

## 描述
用于在游戏页面的顶部或底部显示横幅广告。

## 变量
| 变量名         |   类型      |  作用        |
| ------------- |-------------|-------------|
| urlPC | string | 用于PC设备的广告页面地址 |
| urlMobile | string　| 用于移动设备的广告页面地址 |
| height | number | 横幅广告高度 |
| location | number | 广告显示的位置，BannerAd.AD_LOCATION_BOTTOM（显示在页面底部）和BannerAd.AD_LOCATION_TOP（显示在页面顶部） |

## 方法
| 方法名 | 作用 |
| ------------- |-------------|
| display | 显示横幅广告 |
| hide | 隐藏横幅广告 |
