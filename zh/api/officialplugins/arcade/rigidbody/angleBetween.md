# [qc.arcade.RigidBody](../RigidBody.md).angleBetween

## 原型
* void angleBetween(source, target)

## 参数
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| source | object | 物体1（需要包含x和y属性） |
| target | object | 物体2（需要包含x和y属性） |

## 返回值
| 类型 | 说明 |
| ------------- | -------------|
| number | 两个物体间夹角的弧度值 |

## 说明
计算两个物体之间的夹角
