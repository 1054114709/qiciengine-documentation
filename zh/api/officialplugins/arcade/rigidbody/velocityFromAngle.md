# [qc.arcade.RigidBody](../RigidBody.md).velocityFromAngle

## 原型
* booean velocityFromAngle(angle, speed, point)

## 参数
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| angle | number | 旋转角度 |
| speed | number | 移动速度 |
| point | qc.Point | 水平和垂直方向的速度 |

## 返回值
水平和垂直方向的速度

## 说明
根据角度和速度，得到水平和垂直方向的速度
