# 富文本表情图标注册器

## 描述
将图集引入到富文本中，在富文本中使用[image package='name' frame='frame'/]来显示。

## 变量
| 变量名           | 类型            | 作用            |
| ------------- | ------------- | ------------- |
| packageName   | string        | 注册的包名         |
| packageAtlas  | qc.Atlas      | 表明包图集         |
| richText      | qc.Node       | 注册到的富文本节点     |
