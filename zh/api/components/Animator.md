# qc.Animator  

## 描述
用来播放动作编辑器和动作管理器创建出来的动作文件。  

## 方法  
| 方法名 | 作用 |
| ------------- |-------------|
| [getAction](Animator_getAction.md) | 根据动作名或动作索引取得动作对象。      |
| [play](Animator_play.md) | 播放一个动作。 |
| [stop](Animator_stop.md) | 停止一个动作。 |
