# [qc.LayoutElement](LayoutElement.md).getLayoutElement

## 原型
* object getLayoutElement(node)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| node | [qc.Node](../gameobject/CNode.md) | 节点 |

## 说明
获取一个节点上的布局元素

## 范例
返回的格式如下：
````
    {
        ignoreLayout : NaN,
        minWidth : NaN,
        minHeight : NaN,
        preferredWidth : NaN,
        preferredHeight : NaN,
        flexibleWidth : NaN,
        flexibleHeight : NaN,
    };
````
