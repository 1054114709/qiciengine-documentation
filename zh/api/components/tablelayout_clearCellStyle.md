# [qc.TableLayout](TableLayout.md).clearCellStyle

## 原型
* void clearCellStyle(int column, int row)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| column | int | 列，如果为 null，则表示清除指定行的样式，如果为 null 且row === null,则清除全局样式  |
| row | int | 行，如果为 null，则表示清除指定列的样式，如果为 null 且column === null,则清除全局样式 |

## 说明
清除一个单元格样式
