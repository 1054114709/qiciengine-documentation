# TweenTransform
inherit [Tween](Tween.md)

## 描述
驱动对象位置变化的组件，起始位置和终点位置由两个物体决定

## 变量
| 变量名 | 类型 | 作用 |
| ------------- |-------------|-------------|
| from | [qc.Node](../gameobject/CNode.md) | 决定初始位置的物体 |
| to | [qc.Node](../gameobject/CNode.md) | 决定终点位置的物体 |
