# [qc.Math](README.md).maxAdd

## 原型
* number maxAdd(value, amount, max)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| value | number | 值 |
| amount | number | 加值 |
| max | number | 最大值 |
| 返回值 | number |  |

## 说明
限制最大值的加法操作

## 范例
````
    // 返回 2
    var ret = game.math.maxAdd(1, 3, 2);

    // 返回 4
    var ret = game.math.maxAdd(1, 3, 4);
````
