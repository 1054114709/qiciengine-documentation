# [qc.Math](README.md).distance

## 原型
* number distance(x1, y1, x2, y2)

## 参数说明
| 参数名 | 类型 | 说明 |
| ------------- | ------------- | -------------|
| x1 | number | 点1的x坐标 |
| y1 | number | 点1的y坐标 |
| x2 | number | 点2的x坐标 |
| y2 | number | 点2的y坐标 |
| 返回值 | number | 两点间的距离 |

## 说明
取得两点的距离

## 范例
````
    // 返回 2.23606797749979
    var ret = game.math.distance(1, 2, 0, 4);
````
