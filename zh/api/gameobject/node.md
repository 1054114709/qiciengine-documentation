# [qc.GameObjectFactory](GameObjectFactory.md).node

## 原型
* [qc.Node](CNode.md) node([qc.Node](CNode.md) parent)

## 参数
| 参数名 | 类型 | 说明 |
| ----------- | ----------- | ----------- |
| parent | [qc.Node](CNode.md) | 挂载的父亲对象（允许为空，空的话挂载到根节点） |

## 描述
创建一个空的节点

## 范例
````
    var node = game.add.node(game.world)
````
