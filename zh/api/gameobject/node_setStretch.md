# [qc.Node](CNode.md).setStretch

## 原型
* void setStretch(number left, number right, number top, number bottom)

## 参数
| 参数名 | 类型 |  说明 |
| --------- | --------- | --------- |
| left | number | 离左边的距离 |
| right | number | 离右边的距离 |
| top | number | 离右边的距离 |
| bottom | number | 离右边的距离 |

## 描述
设置节点相对于父亲的位置，以上、下、左、右边距来指定
